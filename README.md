Тестовое задание для Versta
- Backend: ASP.NET Core (.NET10)
- Frontend: ReactJS
- БД: PostgreSQL, ORM: EF Core

Для запуска:
1) Запустить docker, и создать/запустить контейнер с БД через docker compose up -d
2) Локально запустить asp.net core проект (запустится на http://localhost:5102)
3) Через npm run dev запустить фронт
